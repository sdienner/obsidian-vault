# Obsidian Vault — Claude Context

This is Scott's personal Obsidian vault, managed with Claude Code.

## About Scott

Engineering leader managing a team of 11. Focused on Career & Leadership growth. Coaching-oriented: prefers Claude to ask probing questions, push back, and help think through problems — not just execute.

## System Purpose

This vault supports professional effectiveness and leadership work:
- **Career & Leadership** — team management, 1:1s, performance reviews, engineering strategy
- **Projects** — active engineering initiatives
- **Meetings** — meeting notes and follow-ups
- **People** — direct reports and key stakeholders

## The Cascade

Goals flow from long-term vision to daily actions:

```
3-Year Vision (Goals/1. 3-Year Vision.md)
    ↓
Yearly Goals (Goals/2. Yearly Goals.md)
    ↓
Weekly Review (Goals/3. Weekly Review.md)
    ↓
Daily Notes (Daily/YYYY-MM-DD.md)
```

## Conventions

- Weekly review day: **Monday**
- Daily notes: `Daily/YYYY-MM-DD.md`
- Project notes: one file per project in `Projects/`
- Person notes: one file per person in `People/`
- Meeting notes: `Meetings/YYYY-MM-DD Topic.md`
- Templates: `Templates/` — use as starting points, not strict rules
