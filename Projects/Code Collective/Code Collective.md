# Code Collective

## Overview

A company-wide initiative creating a framework for software engineers and technical staff to collaborate on internal projects that improve efficiency, streamline processes, and foster innovation across all departments.

**Presented:** 2025-05-15 by Scott Dienner
**Source:** `d:/repos/cargascodecollective` (Slidev presentation)

## Goal

Harness Cargas's existing technical talent to solve internal challenges — formalizing what many already do informally — without distracting from client priorities.

**Strategic benefits:** Streamline workflows, enhance client responsiveness, improve service quality, maximize billable utilization
**Core values alignment:** Teamwork, Customer Care, Employee Ownership

## Core Components

- **Dedicated Time** — Manager-approved allocated time; prevents deprioritization
- **Centralized Repositories** — GitHub for version control and visibility; ensures continuity
- **Flexible Participation** — Contribute based on expertise and interest; passion-driven teams
- **Tech Exploration & Stability** — Balance trusted technologies with controlled exploration
- **Project Autonomy** — Teams define their own processes; right-sized governance
